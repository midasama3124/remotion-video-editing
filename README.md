## Quick Start for a New Project

1. Create the project folder and `video/` subfolder
2. Add footage following naming conventions (see [01](docs/01_PROJECT_STRUCTURE.md))
3. Copy the phase timing worksheet from [07](docs/07_VIDEO_STRUCTURE_TEMPLATE.md)
4. Analyze the reference video and fill in the EDL segments
5. Build Remotion components following the layer structure in [07](docs/07_VIDEO_STRUCTURE_TEMPLATE.md)
6. Validate layouts against Instagram safe zone (see [02](docs/02_CANVAS_LAYOUT.md))
7. Mix audio layers per [06](docs/06_AUDIO.md) guidelines
8. Render and review
